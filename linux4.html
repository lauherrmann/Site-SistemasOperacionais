Instalação e uso de servidor web no Linux

  1. A instalação do apache é simples e, para isso, basta abrir o terminal e inserir o comando: sudo apt install apache2
  2. Depois de instalar o apache, é necessário ajustar as configurações do Firewall. Basta utilizar o comando: sudo ufw app list
  3. Depois, é necessário escolher o perfil Apache:sudo ufw allow in "Apache"
  4. É possível verificar o estado do Firewall com o comando: sudo ufw status
  5. Por último, é possível verificar se o serviço está operando. Basta abrir o navegador e inserir o endereço da máquina que tem o Ubuntu instalado.
