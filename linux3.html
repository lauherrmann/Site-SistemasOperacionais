Instalação e uso de linguagens C e C++ em modo caractere no Linux

  1. O primeiro passo é baixar as atualizações e informações dos pacotes de todos os repositórios configurados, através do comando: sudo apt update
  2. Após isso, é necessário instalar as atualizações disponíveis dos pacotes instalados atualmente no sistema, através do comando: sudo apt upgrade
  3. Depois disso, deve ser feita a instalação da linguagem C++, através do comando: sudo apt install g++
  4. Também deve ser feita a instalação da linguagem C, através do comando: sudo apt install gcc
  5. Após esses comandos, as instalações das linguagens devem estar concluídas.
