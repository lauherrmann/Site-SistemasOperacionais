Comandos essenciais do Windows

OBS: os comandos devem ser escritos em letras minúsculas

  • MD (nomedodiretorio): cria um diretório.
  • CD (nomedodiretorio): entra no diretório.
  • RD: remove o diretório.
  • DIR: mostra o conteúdo do diretório.
  • CD .. : retorna ao diretório anterior.
  • CD/: volta para a pasta raiz.
  • COPY: faz uma cópia do diretório. Para utilizar esse comando, é necessário digitar o caminho da pasta onde se encontra o arquivo para copiar e também é necessário incluir a pasta de destino.
  • REN: renomeia o diretório.
  • MOVE: move arquivos de um diretório para outro.
