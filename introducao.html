O que é um Sistema Operacional?

Os sistemas operacionais surgem para resolver o problema de comunicação entre o usuário e a máquina. Isso acontece, pois nós, seres humanos, utilizamos linguagens diferentes das utilizadas pelo computador. O computador utiliza a linguagem do um (1) e do zero (0) (código binário), logo, não conseguimos nos comunicar diretamente com ele.

O sistema operacional vai possibilitar a comunicação e, juntamente com o hardware, determina tudo que o computador pode fazer. O sistema operacional é o primeiro programa executado ao ligarmos o computador, portanto é um garantidor de funcionamento, atuando para que um programa não possa interferir no funcionamento de outro e gerenciando outros recursos do computador (como mouse e teclado).

Sem o sistema operacional, os usuários teriam que ter conhecimentos avançados em linguagens e comandos para manipular o computador e, ainda, os softwares teriam que ser desenvolvidos para uma configuração específica de hardware, o que dificultaria o desenvolvimento deles.

Dessa forma, o sistema operacional trabalha de forma assíncrona - ele está em funcionamento desde o momento que o computador é ligado até ser desligado.

Módulo de um SO

Os sistemas operacionais são construídos com vários módulos para torná-los mais fáceis de serem escritos. Cada módulo torna-se responsável por uma função e os módulos típicos normalmente são: núcleo, gerenciador de processo, escalonador e gerenciador de arquivo.

O núcleo é o cérebro do computador, é a ligação entre o processamento dos dados e os programas. Dessa forma, torna-se o elo entre o hardware e software, permitindo a execução dos aplicativos e utilização dos recursos da máquina.

No gerenciador de processo, ocorre a execução de programas, permitindo alocar recursos, compartilhar dados e sincronizar a execução.

O escalonador decide o momento em que cada processo obtém acesso à CPU. Para isso, é utilizado algoritmos que avaliam o cenário que o sistema é utilizado. O sistema operacional decide quando escalonar, intervir na CPU, podendo ser não preemptivo que obriga a tomada de decisão (como a criação de um novo processo) ou preemptivo que escolhe um processo e concede a CPU por um tempo.

O gerenciador de arquivo é utilizado para criar e organizar diretórios no sistema operacional, gerenciando espaços alocados, criando e deletando arquivos e fazendo a manipulação de diretórios.

Windows e Linux

São os dois sistemas operacionais mais famosos. O Windows não possibilita acesso ao código fonte, por conta disso não é possível alterar nada no sistema e é necessário pagar pela sua licença. É também o sistema operacional padrão para computadores com processadores Intel por ter uma interface gráfica agradável e mais conhecida. Sua utilização é de maior facilidade em relação ao Linux, mas ele é mais vulnerável a vírus, mesmo com uma boa configuração.

O Linux proporciona acesso completo ao sistema, já que tem o código fonte aberto e, por ser licenciado, tem distribuição livre. Com esse sistema operacional, pode fazer tudo pela linha de comando, já que ele é dedicado para ela - algumas companhias criaram interface gráfica (como Ubuntu). A exposição a vírus é menor, pois existem pouquíssimos vírus criados para o Linux. Uma desvantagem é a escassez de aplicativos, já que a procura não é feita pela internet e sim no repositório do sistema operacional.
