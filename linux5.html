Liberação de serviço web no firewall no Linux

Para fazer a liberação desse serviço, basta executar os seguintes comandos:

  • sudo ufw app list
  • sudo ufw allow ‘Apache’
  • sudo chmod -R 777 diretorio

Instalação e uso do VS Code para construção de website no Linux

  1. A instalação do VS Code no Linux é muito simples. Para isso, basta acessar o site do Visual Studio Code e fazer o download da última versão, clicando no botão “Download for Linux”.
  2. Após isso, terá um pacote .deb na pasta Downloads. Para isso, abra o terminal e execute o comando: sudo apt install ./ARQUIVO_BAIXADO.deb
  3. Por fim, o VS Code já deve aparecer no menu de aplicativos.
