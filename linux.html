Comandos essenciais do Linux

OBS: os comandos devem estar em letras minúsculas

  • MKDIR (nome do diretório): cria um diretório especificado;
  • CD (nome do diretório): entra no diretório;
  • LS-L: mostra o conteúdo do diretório com detalhes;
  • LS: exibe a lista de diretórios;
  • CD .. : retorna ao diretório anterior
  • RMDIR nomedodiretorio: remove o diretório;
  • CP: copia um arquivo de um diretório para outro;
  • PWD: exibe o caminho do diretório de trabalho;
  • MV: move o arquivo de um diretório para outro.
  • TOUCH : cria um arquivo vazio
